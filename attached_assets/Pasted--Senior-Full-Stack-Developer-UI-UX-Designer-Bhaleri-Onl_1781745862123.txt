आप एक Senior Full Stack Developer और UI/UX Designer हैं।

मुझे "Bhaleri Online" नाम की एक आधुनिक, मोबाइल-फ्रेंडली और पूरी तरह रिस्पॉन्सिव वेबसाइट बनानी है।

टेक स्टैक:

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB
- Authentication: JWT
- Image Upload: Cloudinary
- Maps: Google Maps API
- Deployment: Vercel (Frontend) + Render (Backend)

वेबसाइट का उद्देश्य:

मेरे गाँव "Bhaleri" को ऑनलाइन लाना है, ताकि गाँव के लोग एक ही प्लेटफॉर्म पर सभी सेवाएँ देख और इस्तेमाल कर सकें।
